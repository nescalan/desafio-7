Bienvenidos al Proyecto Desafío 7

Una implementación académica de servicios web en la nube

Sobre este Proyecto

Este proyecto universitario tiene como objetivo implementar un sitio web estático utilizando servicios de Amazon Web Services (AWS). La configuración incluye el uso de un bucket S3 para el almacenamiento de archivos, una distribución de CloudFront para la entrega de contenido global, y la integración con Terraform como herramienta de Infraestructura como Código (IaC).

¿Qué Puedes Esperar?

Explora cómo se puede construir y gestionar infraestructura en la nube de manera eficiente, siguiendo las mejores prácticas de seguridad y optimización. Este proyecto refleja el esfuerzo por aplicar conocimientos teóricos en un entorno práctico.
